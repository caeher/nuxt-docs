# Instalacion

## Requisitos

- `Node.js` compatible con `Nuxt 4`
- `pnpm`

## Instalar dependencias

```bash [Terminal]
pnpm install
```

## Iniciar entorno local

```bash [Terminal]
pnpm dev
```

El portal queda disponible en `http://localhost:3000`.

## Comandos utiles

```bash [Terminal]
pnpm dev
pnpm build
pnpm preview
pnpm lint
pnpm typecheck
```

## Que revisar despues de levantarlo

1. La home del portal en `/`.
2. La navegacion de `Getting Started`.
3. La seccion `/tokenization-api`.
4. Los enlaces internos entre overview y servicios.
5. Los bloques de codigo y tablas markdown.

## Si vas a editar la referencia API

- Revisa primero `tokenization-openapi-json/`.
- Confirma que la pagina markdown refleje rutas, metodos y codigos de respuesta del spec.
- Evita documentar endpoints que no existan en el JSON.

::warning
Este repo todavia conserva algunos rastros de la plantilla original en archivos no funcionales como `README.md`. La documentacion publicada debe guiarse por el contenido de `content/` y por los OpenAPI JSON.
::
