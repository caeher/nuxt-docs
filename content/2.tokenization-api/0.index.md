## Seguridad

Todos los servicios protegidos usan `Authorization: Bearer <access_token>`.

- `X-API-Key`: header opcional en varios endpoints de `tokenization`, `marketplace` y `wallet`.
- `X-2FA-Code`: header adicional para operaciones sensibles como retiros, pagos y desembolsos.
- `422`: error de validacion.
- `401` y `403`: fallos de autenticacion o autorizacion segun el servicio.

::tip
Los endpoints operativos `GET /health`, `GET /ready` y `GET /metrics` existen donde lo indique cada spec. No se asumen rutas fuera del JSON.
::

## Playbooks

### 1. Onboarding + auth + API key

1. Registrar usuario: [`POST /auth/register`](/tokenization-api/auth#registro-post-authregister)
2. Iniciar sesion: [`POST /auth/login`](/tokenization-api/auth#login-post-authlogin)
3. Revisar onboarding: [`GET /auth/onboarding/summary`](/tokenization-api/auth#onboarding-get-authonboardingsummary)
4. Activar 2FA: [`POST /auth/2fa/enable`](/tokenization-api/auth#2fa-post-auth2faenable) y [`POST /auth/2fa/verify`](/tokenization-api/auth#2fa-post-auth2faverify)
5. Crear API key: [`POST /auth/api-keys`](/tokenization-api/auth#api-keys-post-authapi-keys)

```python [Python]
import requests

BASE_URL = "https://api.ecosystem.com"

register = requests.post(
    f"{BASE_URL}/auth/register",
    json={
        "email": "user@example.com",
        "password": "SecretPassword123",
        "display_name": "Alice",
        "referrer_code": None,
    },
    timeout=30,
)
tokens = register.json()["tokens"]

api_key = requests.post(
    f"{BASE_URL}/auth/api-keys",
    headers={"Authorization": f"Bearer {tokens['access_token']}"},
    json={"name": "server-prod", "scopes": ["wallet:read"], "expires_at": None},
    timeout=30,
)
print(api_key.json())
```

```ts [TypeScript]
const baseUrl = 'https://api.ecosystem.com'

const registerRes = await fetch(`${baseUrl}/auth/register`, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    email: 'user@example.com',
    password: 'SecretPassword123',
    display_name: 'Alice',
    referrer_code: null
  })
})

const registerData = await registerRes.json()
const accessToken = registerData.tokens.access_token as string

const apiKeyRes = await fetch(`${baseUrl}/auth/api-keys`, {
  method: 'POST',
  headers: {
    Authorization: `Bearer ${accessToken}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'server-prod',
    scopes: ['wallet:read'],
    expires_at: null
  })
})

console.log(await apiKeyRes.json())
```

### 2. Asset upload -> evaluate -> tokenize

1. Crear activo o subir PDF: [`POST /assets`](/tokenization-api/tokenization#crear-activo-post-assets) o [`POST /assets/upload`](/tokenization-api/tokenization#subir-activo-post-assetsupload)
2. Pedir evaluacion IA: [`POST /assets/{asset_id}/evaluate`](/tokenization-api/tokenization#evaluar-activo-post-assetsasset_idevaluate)
3. Ver detalle: [`GET /assets/{asset_id}`](/tokenization-api/tokenization#detalle-get-assetsasset_id)
4. Tokenizar: [`POST /assets/{asset_id}/tokenize`](/tokenization-api/tokenization#tokenizar-activo-post-assetsasset_idtokenize)

```python [Python]
import requests

BASE_URL = "https://api.ecosystem.com"
headers = {"Authorization": "Bearer <access_token>"}

asset = requests.post(
    f"{BASE_URL}/assets",
    headers=headers,
    json={
        "name": "Warehouse #12",
        "description": "Industrial property",
        "category": "real_estate",
        "valuation_sat": 250000000,
        "documents_url": "https://example.com/docs/warehouse-12.pdf",
    },
    timeout=30,
).json()["asset"]

requests.post(f"{BASE_URL}/assets/{asset['id']}/evaluate", headers=headers, timeout=30)

tokenized = requests.post(
    f"{BASE_URL}/assets/{asset['id']}/tokenize",
    headers={**headers, "Content-Type": "application/json"},
    json={
        "ticker": "WH12",
        "liquid_asset_id": None,
        "taproot_asset_id": None,
        "total_supply": 1000,
        "unit_price_sat": 250000,
        "visibility": "public",
    },
    timeout=30,
)
print(tokenized.json())
```

```ts [TypeScript]
const baseUrl = 'https://api.ecosystem.com'
const headers = { Authorization: 'Bearer <access_token>', 'Content-Type': 'application/json' }

const assetRes = await fetch(`${baseUrl}/assets`, {
  method: 'POST',
  headers,
  body: JSON.stringify({
    name: 'Warehouse #12',
    description: 'Industrial property',
    category: 'real_estate',
    valuation_sat: 250000000,
    documents_url: 'https://example.com/docs/warehouse-12.pdf'
  })
})

const assetData = await assetRes.json()
const assetId = assetData.asset.id as string

await fetch(`${baseUrl}/assets/${assetId}/evaluate`, {
  method: 'POST',
  headers: { Authorization: 'Bearer <access_token>' }
})

const tokenizeRes = await fetch(`${baseUrl}/assets/${assetId}/tokenize`, {
  method: 'POST',
  headers,
  body: JSON.stringify({
    ticker: 'WH12',
    liquid_asset_id: null,
    taproot_asset_id: null,
    total_supply: 1000,
    unit_price_sat: 250000,
    visibility: 'public'
  })
})

console.log(await tokenizeRes.json())
```

### 3. Order -> escrow -> settlement or dispute

1. Crear orden: [`POST /orders`](/tokenization-api/marketplace#crear-orden-post-orders)
2. Ver orderbook: [`GET /orderbook/{token_id}`](/tokenization-api/marketplace#orderbook-get-orderbooktoken_id)
3. Consultar escrow: [`GET /escrows/{trade_id}`](/tokenization-api/marketplace#escrow-get-escrowstrade_id)
4. Firmar liberacion: [`POST /escrows/{trade_id}/sign`](/tokenization-api/marketplace#firmar-escrow-post-escrowstrade_idsign)
5. Si aplica, abrir disputa: [`POST /trades/{trade_id}/dispute`](/tokenization-api/marketplace#abrir-disputa-post-tradestrade_iddispute)

```python [Python]
import requests

BASE_URL = "https://api.ecosystem.com"
headers = {"Authorization": "Bearer <access_token>", "Content-Type": "application/json"}

order = requests.post(
    f"{BASE_URL}/orders",
    headers=headers,
    json={
        "token_id": "tok_123",
        "side": "buy",
        "order_type": "limit",
        "quantity": 10,
        "price_sat": 50000,
        "trigger_price_sat": None,
    },
    timeout=30,
)
print(order.json())
```

