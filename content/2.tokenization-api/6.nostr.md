# Nostr Service

Spec fuente: `tokenization-openapi-json/nostr.json` (`info.version = 0.1.0`).

## Detalle por endpoint

### Listar campanas (`GET /campaigns`)

- Proposito: listar campanas zap del usuario autenticado.
- Auth requerida: Bearer.
- Parametros: sin path ni query.
- Respuesta: `200` con arreglo de campanas.

### Crear campana (`POST /campaigns`)

- Proposito: crear una campana Nostr.
- Auth requerida: Bearer.
- Body JSON: `name`, `funding_mode`, `reward_amount_sat`, `budget_total_sat`, `max_rewards_per_user`, `start_at`, `end_at`, `triggers`.
- Respuesta: `201` con la campana creada; `422` si el payload no valida.

```python [Python]
import requests

response = requests.post(
    "https://api.plan-b2b-flow.cuboplus.caeher.com/v1/campaigns",
    headers={
        "Authorization": "Bearer <access_token>",
        "Content-Type": "application/json",
    },
    json={
        "name": "Spring campaign",
        "funding_mode": "intraledger",
        "reward_amount_sat": 100,
        "budget_total_sat": 10000,
        "max_rewards_per_user": 3,
        "start_at": None,
        "end_at": None,
        "triggers": ["note", "repost"],
    },
    timeout=30,
)
print(response.json())
```

```ts [TypeScript]
const response = await fetch('https://api.plan-b2b-flow.cuboplus.caeher.com/v1/campaigns', {
  method: 'POST',
  headers: {
    Authorization: 'Bearer <access_token>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Spring campaign',
    funding_mode: 'intraledger',
    reward_amount_sat: 100,
    budget_total_sat: 10000,
    max_rewards_per_user: 3,
    start_at: null,
    end_at: null,
    triggers: ['note', 'repost']
  })
})

console.log(await response.json())
```

### Detalle (`GET /campaigns/{campaign_id}`)

- Proposito: obtener una campana concreta.
- Auth requerida: Bearer.
- Path: `campaign_id`.
- Respuesta: `200` con la campana; `422` si el identificador no valida.

### Fondeo (`POST /campaigns/{campaign_id}/fund/intraledger`, `POST /campaigns/{campaign_id}/fund/external`)

- Proposito: reservar balance interno o generar invoice externa para financiar una campana.
- Auth requerida: Bearer.
- Path: `campaign_id`.
- Body JSON en ambos casos: `amount_sat`.
- Respuestas:
  - `fund/intraledger`: `200` con campana actualizada.
  - `fund/external`: `200` con datos de funding, incluyendo `payment_hash` y `payment_request`.

### Estado (`POST /campaigns/{campaign_id}/activate`, `POST /campaigns/{campaign_id}/pause`, `POST /campaigns/{campaign_id}/cancel`)

- Proposito: activar, pausar o cancelar una campana.
- Auth requerida: Bearer.
- Path: `campaign_id`.
- Body: ninguno.
- Respuesta: `200` con campana actualizada.

```python [Python]
import requests

response = requests.post(
    "https://api.plan-b2b-flow.cuboplus.caeher.com/v1/campaigns/cmp_123/activate",
    headers={"Authorization": "Bearer <access_token>"},
    timeout=30,
)
print(response.json())
```

```ts [TypeScript]
const response = await fetch('https://api.plan-b2b-flow.cuboplus.caeher.com/v1/campaigns/cmp_123/activate', {
  method: 'POST',
  headers: { Authorization: 'Bearer <access_token>' }
})

console.log(await response.json())
```

### Coincidencias y payouts (`GET /campaigns/{campaign_id}/matches`, `GET /campaigns/{campaign_id}/payouts`)

- Proposito: auditar interacciones detectadas y payouts intentados.
- Auth requerida: Bearer.
- Path: `campaign_id`.
- Respuestas:
  - `GET /matches`: `200` con arreglo de matches.
  - `GET /payouts`: `200` con arreglo de payouts.

### Anuncios (`POST /announcements`)

- Proposito: publicar un anuncio clasificado en relays Nostr.
- Auth requerida: Bearer.
- Body JSON: `title`, `content`, `summary`, `identifier`, `hashtags`, `location`, `price_amount`, `price_currency`, `price_frequency`, `reference_url`, `image_urls`, `status`.
- Respuesta: `201` con `id`, `kind`, `pubkey`, `identifier`, `accepted_relays`, `failed_relays`.

```python [Python]
import requests

response = requests.post(
    "https://api.plan-b2b-flow.cuboplus.caeher.com/v1/announcements",
    headers={
        "Authorization": "Bearer <access_token>",
        "Content-Type": "application/json",
    },
    json={
        "title": "Token sale",
        "content": "Industrial property fractions available",
        "summary": "New listing",
        "identifier": "listing-001",
        "hashtags": ["rwa", "bitcoin"],
        "location": None,
        "price_amount": 1000,
        "price_currency": "USD",
        "price_frequency": None,
        "reference_url": "https://example.com/listings/1",
        "image_urls": [],
        "status": "published",
    },
    timeout=30,
)
print(response.json())
```

