# Tokenization Service

Spec fuente: `tokenization-openapi-json/tokenization.json` (`info.version = 0.1.0`).

## Detalle por endpoint

### Catalogo (`GET /assets`)

- Proposito: devolver catalogo filtrado de activos.
- Auth requerida: Bearer; header opcional `X-API-Key`.
- Query: `status`, `category`, `cursor`, `limit`.
- Respuesta: `200` con `assets` y `next_cursor`; `422` si los filtros no validan.

### Crear activo (`POST /assets`)

- Proposito: registrar un activo para revision.
- Auth requerida: Bearer; header opcional `X-API-Key`.
- Body JSON: `name`, `description`, `category`, `valuation_sat`, `documents_url`.
- Respuesta: `201` con `asset`; `422` si el payload no valida.

```python [Python]
import requests

response = requests.post(
    "https://api.ecosystem.com/assets",
    headers={
        "Authorization": "Bearer <access_token>",
        "Content-Type": "application/json",
    },
    json={
        "name": "Warehouse #12",
        "description": "Industrial property",
        "category": "real_estate",
        "valuation_sat": 250000000,
        "documents_url": "https://example.com/docs/warehouse-12.pdf",
    },
    timeout=30,
)
print(response.json())
```

```ts [TypeScript]
const response = await fetch('https://api.ecosystem.com/assets', {
  method: 'POST',
  headers: {
    Authorization: 'Bearer <access_token>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Warehouse #12',
    description: 'Industrial property',
    category: 'real_estate',
    valuation_sat: 250000000,
    documents_url: 'https://example.com/docs/warehouse-12.pdf'
  })
})

console.log(await response.json())
```

### Subir activo (`POST /assets/upload`)

- Proposito: crear activo con documento PDF gestionado por el servicio.
- Auth requerida: Bearer; header opcional `X-API-Key`.
- Body `multipart/form-data`: `name`, `description`, `category`, `valuation_sat`, `document`.
- Respuesta: `201` con `asset`; `422` en validacion fallida.

```python [Python]
import requests

with open("asset.pdf", "rb") as document:
    response = requests.post(
        "https://api.ecosystem.com/assets/upload",
        headers={"Authorization": "Bearer <access_token>"},
        data={
            "name": "Warehouse #12",
            "description": "Industrial property",
            "category": "real_estate",
            "valuation_sat": "250000000",
        },
        files={"document": ("asset.pdf", document, "application/pdf")},
        timeout=30,
    )
print(response.json())
```

```ts [TypeScript]
const formData = new FormData()

```

### Evaluar activo (`POST /assets/{asset_id}/evaluate`)

- Proposito: lanzar evaluacion IA sobre un activo propio.
- Auth requerida: Bearer; header opcional `X-API-Key`.
- Path: `asset_id`.
- Body: ninguno.
- Respuesta: `202` con `message` y `estimated_completion`; `422` en errores de validacion.

### Tokenizar activo (`POST /assets/{asset_id}/tokenize`)

- Proposito: tokenizar un activo aprobado.
- Auth requerida: Bearer; header opcional `X-API-Key`.
- Path: `asset_id`.
- Body JSON: `ticker`, `liquid_asset_id`, `taproot_asset_id`, `total_supply`, `unit_price_sat`, `visibility`.
- Respuesta: `201` con `asset`; `422` si el activo no esta listo o el body no valida.

```python [Python]
import requests

response = requests.post(
    "https://api.ecosystem.com/assets/asset_123/tokenize",
    headers={
        "Authorization": "Bearer <access_token>",
        "Content-Type": "application/json",
    },
    json={
        "ticker": "WH12",
        "liquid_asset_id": None,
        "taproot_asset_id": None,
        "total_supply": 1000,
        "unit_price_sat": 250000,
        "visibility": "public",
    },
    timeout=30,
)
print(response.json())
```

```ts [TypeScript]
const response = await fetch('https://api.ecosystem.com/assets/asset_123/tokenize', {
  method: 'POST',
  headers: {
    Authorization: 'Bearer <access_token>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    ticker: 'WH12',
    liquid_asset_id: null,
    taproot_asset_id: null,
    total_supply: 1000,
    unit_price_sat: 250000,
    visibility: 'public'
  })
})

console.log(await response.json())
```
