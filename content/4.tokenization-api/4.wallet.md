El **Wallet Service** es la interfaz financiera multired (On-chain, Lightning y Liquid).

## Referencia de Endpoints

### 1. Saldos y Cuentas

| Método | Endpoint | Descripción |
| :--- | :--- | :--- |
| `GET` | `/wallet` | Saldo consolidado (BTC y RWA) y dirección. |
| `GET` | `/wallet/transactions` | Historial completo de movimientos. |
| `GET` | `/wallet/yield/summary` | Resumen de rendimientos acumulados. |
| `GET` | `/wallet/custody` | Estado de custodia y seguridad. |

#### Ejemplo: Saldo
```bash [cURL]
curl -X GET https://api.ecosystem.com/wallet \
  -H "Authorization: Bearer <token>"
```

---

### 2. Lightning Network

| Método | Endpoint | Descripción |
| :--- | :--- | :--- |
| `POST` | `/lightning/invoices` | Crea factura BOLT11 para recibir. |
| `GET` | `/lightning/invoices/{r_hash}` | Consulta estado de una factura. |
| `POST` | `/lightning/payments` | Paga una factura BOLT11 (2FA). |
| `POST` | `/lightning/decode` | Decodifica una factura BOLT11 externa. |

#### Ejemplo: Pagar Factura
```bash [cURL]
curl -X POST https://api.ecosystem.com/lightning/payments \
  -H "Authorization: Bearer <token>" \
  -H "X-2FA-Code: 123456" \
  -d '{"bolt11": "lnbc1..."}'
```

---

### 3. On-chain y Liquid

| Método | Endpoint | Descripción |
| :--- | :--- | :--- |
| `POST` | `/wallet/onchain/address` | Genera nueva dirección Segwit de depósito. |
| `POST` | `/wallet/onchain/withdraw` | Envía BTC a dirección externa (2FA). |
| `GET` | `/wallet/pegin/address` | Dirección para mover BTC a Liquid. |
| `GET` | `/wallet/onchain/fees` | Estimación de comisiones de red. |

---

### 4. Utilidades y Salud

| Método | Endpoint | Descripción |
| :--- | :--- | :--- |
| `GET` | `/wallet/qr` | Genera imagen PNG de código QR. |
| `GET` | `/wallet/fiat/onramp/providers` | Lista proveedores de compra fiat. |
| `POST` | `/wallet/fiat/onramp/session` | Inicia sesión de compra externa. |
| `GET` | `/metrics` | Métricas operativas. |
| `GET` | `/health` | Liveness check. |
| `GET` | `/ready` | Readiness check. |

---

## Modelos de Datos (Parámetros)

### Retiro On-chain (`POST /wallet/onchain/withdraw`)
| Campo | Tipo | Descripción |
| :--- | :--- | :--- |
| `destination` | string | Dirección Bitcoin destino. |
| `amount_sat` | integer | Monto en satoshis. |
| `fee_rate` | number | Tasa opcional (sat/vB). |

::tip
Usa `/wallet/qr?value=DEPOSITO` para obtener un QR renderizado por el servidor.
::
