El **Marketplace Service** facilita el intercambio de tokens RWA mediante un motor de emparejamiento (orderbook) y un sistema de custodia segura (escrow).

## Referencia de Endpoints

### 1. Órdenes y Mercado

| Método | Endpoint | Descripción |
| :--- | :--- | :--- |
| `POST` | `/orders` | Coloca una orden limit (compra/venta). |
| `GET` | `/orders` | Lista órdenes propias (activas/históricas). |
| `GET` | `/orderbook/{token_id}` | Muestra Bids y Asks para un token específico. |
| `DELETE` | `/orders/{order_id}` | Cancela una orden abierta. |

#### Ejemplo: Nueva Orden
```bash [cURL]
curl -X POST https://api.ecosystem.com/orders \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "token_id": "UUID",
    "side": "buy",
    "order_type": "limit",
    "quantity": 10,
    "price_sat": 50000
  }'
```

---

### 2. Escrows y Trades

| Método | Endpoint | Descripción |
| :--- | :--- | :--- |
| `GET` | `/trades` | Lista intercambios ejecutados exitosamente. |
| `GET` | `/escrows/{trade_id}` | Detalle del estado de firmas de un trade. |
| `POST` | `/escrows/{trade_id}/sign` | Firma para liberar/reembolsar (2FA). |
| `POST` | `/escrows/{trade_id}/external-sign` | Carga firmas generadas externamente. |

#### Ejemplo: Firma de Escrow
```bash [cURL]
curl -X POST https://api.ecosystem.com/escrows/{id}/sign \
  -H "Authorization: Bearer <token>" \
  -H "X-2FA-Code: 123456" \
  -d '{"action": "release"}'
```

---

### 3. Disputas y Utilidades

| Método | Endpoint | Descripción |
| :--- | :--- | :--- |
| `POST` | `/trades/{trade_id}/dispute` | Abre una disputa sobre un intercambio. |
| `POST` | `/trades/{trade_id}/dispute/resolve` | Resuelve disputa por acuerdo mutuo. |
| `GET` | `/metrics` | Métricas de Prometheus. |
| `GET` | `/health` | Estado de salud. |
| `GET` | `/ready` | Disponibilidad del servicio. |

---

## Modelos de Datos (Parámetros)

### Órdenes (`POST /orders`)
| Campo | Tipo | Descripción |
| :--- | :--- | :--- |
| `token_id` | string | UUID del token RWA. |
| `side` | string | `buy` o `sell`. |
| `price_sat` | integer | Precio unitario en satoshis. |

### Escrows
- **Acciones**: `release` (liberar al vendedor), `refund` (devolver al comprador).
- **Seguridad**: Requiere header `X-2FA-Code`.

::warning
Las órdenes de tipo `market` no están disponibles actualmente.
::
